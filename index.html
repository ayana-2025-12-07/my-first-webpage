Интернет және веб-технологиялар

Бұл веб-бет HTML, CSS және JavaScript технологияларын қолдану арқылы жасалған қарапайым мысал болып табылады.

HTML — беттің құрылымын, CSS — безендірілуін, ал JavaScript — бетке динамикалық мінез-құлық қосады. Осы бетте біз солардың негізгі мүмкіндіктерін көрсеттік.

Бұл бетте қолданылған технологиялар

  • HTML — тақырыптар, абзацтар, тізімдер, сілтемелер;
  • CSS — мәтіннің түсі, фон, шрифттер, блоктардың орналасуы;
  • JavaScript — батырма арқылы мәтінді өзгерту және форма тексеру.
